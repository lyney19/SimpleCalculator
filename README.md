# Simple JavaFX Calculator

## 📌 Description
Simple calculator with a graphical interface written in Java using JavaFX.
The project was created to study UI and application architecture.

## 🚀 Features
- Basic operation: +, -, *, /
- Execution history
- Clear functions (C, AC, backspace)
- Support for decimal numbers

## 🛠 Tech Stack
- Java
- JavaFX


## ⚙️ Installation & Run
1. Clone repository:
```bash
git clone https://github.com/your-username/calculator.git
```
1. Open in IDE (IntelliJ IDEA, Eclipse)
2. Run `Main.java`

## 📁 Project Structure
```
SimpleCalculator/
├─ Main.java            # UI
├─ CalculatorController # Controller
├─ Calculator           # Model
└─ Operation.java       # Model
```
